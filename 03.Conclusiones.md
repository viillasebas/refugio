## Conclusión

## Podemos concluir que el desarrollo de este sistema demostró cómo la tecnología con propósito puede transformar la gestión de un refugio de animales, pasando del desorden manual a una plataforma ágil, segura y centralizada que facilita el control médico y las adopciones. Además, el proceso nos enseñó que el ciclo de vida del software va mucho más allá de escribir código; diseñar un plan de pruebas robusto garantizó la estabilidad del sistema, mientras que estructurar una estrategia de soporte y backups aseguró su continuidad en el futuro. Finalmente, el uso de Git y GitHub fue la pieza clave para coordinar nuestro trabajo en equipo, permitiéndonos colaborar de forma ordenada y simular con éxito la dinámica de un entorno laboral profesional.
