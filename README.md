# Refugio de Animales (Projecto Academico)

## Objetivo

Desarrollar un sistema de información centralizado para un refugio de animales que permita gestionar de forma organizada y segura el registro de mascotas, el control de adoptantes, el procesamiento de solicitudes de adopción y el seguimiento de historias clínicas básicas.

## Tecnologías de Desarrollo

* **Frontend:** HTML, CSS, JavaScript (React) 
* **Backend:** Node.js / Express
* **Base de Datos:** MySQL
* **Control de Versiones:** Git / GitHub

## Cómo Ejecutar (Conceptual)

1. Clonar el repositorio de GitHub.
2. Instalar las dependencias del proyecto.
3. Iniciar el servidor del backend y la base de datos.
4. Ejecutar la aplicación de cara al usuario (frontend).
